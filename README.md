# 幹話產生器
一個利用node.js及express做成的網頁幹話產生器

## 安裝

1.打開你的Termianl，複製此專案至本機電腦 https://github.com/hijerry1007/nonesenseGenerator.git
2.開啟終端機，進入存放此專案的資料夾restaurant
3.安裝npm在Terminal輸入npm install指令
4.安裝nodemon套件在 Termianl輸入 nodemon app.js指令
5.輸入本機地址localhost:3000 即可瀏覽此頁面
6.選擇所要的職業並點擊產生幹話按鈕即可產生幹話

## 功能列表

1. 產生工程師的幹話
2. 產生設計師的幹話
3. 產生創業者的幹話
